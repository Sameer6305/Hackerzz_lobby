## üîÑ User Flow

### Registration Process

```
1. User visits /register
   ‚Üì
2. Fills registration form:
   - Full Name *
   - Email Address *
   - College/University
   - Phone Number
   - Password (min 6 chars) *
   - Confirm Password *
   ‚Üì
3. System validates:
   ‚úì All required fields filled
   ‚úì Valid email format
   ‚úì Password meets requirements
   ‚úì Passwords match
   ‚úì Email not already registered
   ‚Üì
4. Create user account:
   - Generate unique user ID
   - Store user data
   - Create default profile
   - Save to localStorage
   ‚Üì
5. Success ‚Üí Redirect to /signin
```

### Sign-In Process

```
1. User visits /signin
   ‚Üì
2. Enters credentials:
   - Email
   - Password
   ‚Üì
3. System validates:
   ‚úì Email exists in database
   ‚úì Password matches
   ‚Üì
4. Create session:
   - Set currentUser in localStorage
   - Load user's profile data
   - Dispatch sign-in event
   ‚Üì
5. Success ‚Üí Redirect to /dashboard
```

### Protected Page Access

```
1. User tries to access /dashboard
   ‚Üì
2. ProtectedRoute checks authentication
   ‚Üì
3. If NOT authenticated:
   ‚Üí Redirect to /signin
   ‚Üì
4. If authenticated:
   ‚Üí Load page
   ‚Üí Display user's data
```

### ‚ö†Ô∏è Production Recommendations

**IMPORTANT**: This implementation is for development/demo purposes. For production:

1. **Password Security**
   ```javascript
   // DO NOT store plain text passwords
   // Use bcrypt or similar hashing:
   const hashedPassword = await bcrypt.hash(password, 10);
   ```

2. **Backend Integration**
   ```javascript
   // Replace localStorage with API calls:
   const response = await fetch('/api/auth/register', {
     method: 'POST',
     body: JSON.stringify(userData)
   });
   ```

3. **JWT Tokens**
   ```javascript
   // Use JWT for session management:
   const token = jwt.sign({ userId }, SECRET_KEY, { expiresIn: '24h' });
   ```

4. **HTTPS Only**
   - Enforce SSL/TLS
   - Secure cookie flags
   - HttpOnly cookies

5. **Rate Limiting**
   - Prevent brute force attacks
   - Login attempt limits

## üõ†Ô∏è API Reference

### authUtils.js Functions

#### `registerUser(userData)`
Registers a new user account.

**Parameters:**
```javascript
{
  name: string (required),
  email: string (required),
  password: string (required, min 6 chars),
  college: string (optional),
  phone: string (optional)
}
```

**Returns:**
```javascript
{
  success: boolean,
  message: string,
  user: { id, name, email } | undefined
}
```

**Example:**
```javascript
const result = await registerUser({
  name: "John Doe",
  email: "john@example.com",
  password: "password123",
  college: "MIT"
});

if (result.success) {
  console.log("Registration successful!");
}
```

---

#### `signInUser(email, password)`
Authenticates a user and creates a session.

**Parameters:**
```javascript
email: string (required)
password: string (required)
```

**Returns:**
```javascript
{
  success: boolean,
  message: string,
  user: { id, name, email } | undefined
}
```

**Example:**
```javascript
const result = await signInUser("john@example.com", "password123");

if (result.success) {
  navigate('/dashboard');
}
```

---

#### `signOutUser()`
Ends the current user session.

**Returns:**
```javascript
{
  success: boolean,
  message: string
}
```

**Example:**
```javascript
const result = signOutUser();
if (result.success) {
  navigate('/signin');
}
```

---

#### `getCurrentUser()`
Gets the currently signed-in user.

**Returns:**
```javascript
{
  userId: string,
  email: string,
  name: string,
  signedInAt: string (ISO date)
} | null
```

**Example:**
```javascript
const currentUser = getCurrentUser();
if (currentUser) {
  console.log(`Welcome ${currentUser.name}!`);
}
```

---

#### `isAuthenticated()`
Checks if a user is currently signed in.

**Returns:** `boolean`

**Example:**
```javascript
if (isAuthenticated()) {
  // Show dashboard
} else {
  // Redirect to sign in
}
```

---

#### `updateUserProfile(profileData)`
Updates the current user's profile.

**Parameters:**
```javascript
{
  name: string,
  email: string,
  phone: string,
  college: string,
  ...
}
```

**Returns:**
```javascript
{
  success: boolean,
  message: string
}
```
